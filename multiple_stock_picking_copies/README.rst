=============================
Multiple Stock Picking Copies
=============================

This module allows you to set how many copies of a stock picking report you want to print.

Configuration
=============

Install the module, set the number of copies on the field created on the "General Informations" page, and print the picking with the button "Print picking list" (the button appears when the state is Ready to trasnfer or Transferred).
