==================
Qu Iban Converter
==================

Conversión de CCC to IBAN, muestra log del proceso.


Installation
============

Este módulo requiere de los módulos:
	sales_team
	account


Usage
=====

El punto de menú esta en:
      Ventas -> Configuración -> Cuentas Bancarias -> Conversión CCC to IBAN
